Aqui estan algunos de los comandos mas utilizados en la terminal

Comando Descripcion
ls Este comando se utiliza para listar los archivos y directorios en la ubicación actual.
cd (directorio) Este comando se utiliza para cambiar de directorio.
cd .. Este comando se utiliza para retroceder un directorio.
mkdir (nombre) Este comando se utiliza para crear un nuevo directorio.
touch (nombre) Este comando se utiliza para crear un nuevo archivo.
cat (nombre) Este comando se utiliza para ver el contenido de un archivo.
less Este comando se utiliza para ver el contenido de un archivo de forma paginada.
head Este comando se utiliza para ver las primeras lineas de un archivo.
tail Este comando se utiliza para ver las ultimas lineas de un archivo.
vim (nombre) Este comando se utiliza para editar un archivo.
rm (nombre) Este comando se utiliza para eliminar un archivo.
rm -r (nombre) Este comando se utiliza para eliminar un directorio.
rm -rf Este comando se utiliza para eliminar un directorio y su contenido de forma recursiva.
cp (archivo) (destino) Este comando se utiliza para copiar un archivo a una ubicación específica.
mv (archivo) (destino) Este comando se utiliza para mover un archivo a una ubicación específica.
pwd Este comando se utiliza para mostrar la ubicación actual del directorio.
clear Este comando se utiliza para limpiar la pantalla de la terminal.
history Este comando se utiliza para mostrar el historial de comandos utilizados.
