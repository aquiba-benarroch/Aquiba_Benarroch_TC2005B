¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño que se utiliza para dar estilo a los elementos de una página web.
CSS tiene selectores, propiedades y valores que se utilizan para aplicar estilos a los elementos HTML.
Los selectores son los elementos a los que se les aplican los estilos. Los selectores pueden ser de tres tipos:
  • Tipo: Aplican a todos los elementos de ese tipo. Por ejemplo: p
  • Clase: Aplican a todos los elementos con esa clase. Por ejemplo: .Nombre_de_clase
  • ID: Aplican a un solo elemento con ese id. Por ejemplo: #id
Las propiedades son los estilos que se aplican a los elementos. Por ejemplo: color, tamaño de fuente, margen, etc.
Los valores son los valores que se aplican a las propiedades. Por ejemplo: color: red.
